data-container (v2.3.0+)

指定Ajax加载或者内部过场效果的容器。

属性值

为容器元素的id值。

使用

例如，页面上有如下HTML：

<div id="container"></div>
<a href="ajax-page.html" data-container="container">点击ajax加载</a>

则请求的页面会被append到id为container的这个元素内部，而不是默认的document.body的内部。

进阶使用

当data-container和data-classpage同时存在且值有效的时候，可以实现局部过场效果。具体参见API: data-classPage.
